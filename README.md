# reverie-enlive

Render function for reverie/CMS that supports enlive.

## Usage

```clojure
   [reverie-enlive "0.2.0]
```

```clojure

(ns your-site.init
  (:require
    ;; imports here
    [reverie.batteries.enlive]
    ;; more imports here...
    ))
```
